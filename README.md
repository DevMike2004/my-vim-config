# my-vim-config
First vim configuration and was used to get myself comfortable with a temrinal editor. 
Wasn't really sure how to configure in the beginning so I used lazy.nvim as a plugin manager.
It made it very easy to install plugins.
Main points are treesitter for syntax highlighting, telescope for fuzzy find and a replacement for NetRW, 
Mason for different language functionalities like formatting and LSP as well as the built in nvim lsp for 
cmp and debugging purposes. A few smaller plugins were included like flash.nvim, luasnip, and cmp for a seamless experience.

The configuration may be a bit jumbled but I plan to revise it in the future when I know more LUA.
